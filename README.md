## AI-Powered Fraud Detection System

Developed an end-to-end machine learning web application capable of detecting fraudulent financial transactions in real time. The project uses a Random Forest Classifier trained on over 6 million transaction records to identify suspicious transaction patterns with high precision and strong fraud detection performance.

The system includes:
- Data preprocessing and feature engineering using Pandas and Scikit-learn
- Label Encoding for categorical transaction types
- Machine learning model training, evaluation, and prediction
- Flask backend integration for real-time AI inference
- Responsive frontend developed using HTML and CSS
- Deployment on Render for public access

### Key Features
- Real-time fraud prediction
- Web-based user interface
- Trained ML model integration
- Public cloud deployment
- Financial transaction analysis

### Technologies Used
Python, Flask, Pandas, Scikit-learn, Joblib, HTML, CSS, Random Forest Classifier, Render, GitHub
